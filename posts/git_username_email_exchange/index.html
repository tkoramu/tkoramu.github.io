Git_username_email_exchange

メールアドレスとアカウントを全て変更するためのもの

$ git filter-branch -f –env-filter “GIT_AUTHOR_NAME=‘tkoramu’;

GIT_AUTHOR_EMAIL=‘tkoramu@users.noreply.github.com’; GIT_COMMITTER_NAME=‘tkoramu’; GIT_COMMITTER_EMAIL=‘tkoramu@users.noreply.github.com’;” HEAD

下記コマンドを実行すると、「rejectedエラー」が解消される

$ git fetch && git merge origin/master

下記コマンドを実行すると、「rejectedエラー,mergeエラー」が解消される

$ git fetch && git merge –allow-unrelated-histories origin/master
