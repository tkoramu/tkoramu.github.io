Version Up For Git

Git バージョンアップ方法 Ver.CentOS

事前準備

  • wget gcc コマンドがインストールされていない場合 yum -y install wget gcc

古いGitがあれば下記を実行

yum remove git

下記をインストール

sudo yum install curl-devel expat-devel gettext-devel openssl-devel zlib-devel perl-ExtUtils-MakeMaker

ソースファイルをダウンロード

wget https://www.kernel.org/pub/software/scm/git/git-2.9.3.tar.gz

ダウンロードしたファイルを解凍

tar -zxf git-2.2.0.tar.gz

Gitをインストール

cd git-2.9.3
make prefix=/usr/local all
make prefix=/usr/local install

Git バージョン確認

git --version

No such file or directory または そのようなファイルやディレクトリはありません と表示された場合

hash -r
